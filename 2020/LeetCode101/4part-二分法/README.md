## 算法解释  

二分查找也称为二分法或者折半查找，每次取一半查找。将大大减少时间复杂度，对于长度为O(n)的数组，二分法查找复杂度为O(logn)

## 边界问题  
1. 先按照全是等号或者左闭右开写
2. 最后单独考虑等号问题

## 区分双指针  
二分法可以说是双指针特殊情况，但我们一般会将二者区分。双指针的题，通常是一步一步移动，而二分查找则是每次查一半的距离。
